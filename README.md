## Installation :
clone the repo and run install.sh
```
git clone https://github.com/shinas101/betterlockscreen-theme.git
cd betterlockscreen-theme
chmod +x install.sh
./install.sh -h # h for help
			 -b # b for blur
			 -n # n fot noblur
			 
	eg: ./install.sh -n
		
		Please select your theme (No-blur) :
		1) Moonknight-noblur
		2) Baruto-noblur
		3) Solo-leveling-noblur
		4) exit
		#?  
```
